POGO Unplugged

Raspberry Pi that automatically plays Pokemon Go

  • Hardware: Raspberry Pi | Software: Python
  • We built a car that can venture across the Waterloo Campus and collect Pokemon Go rewards for you
  • This car was controlled by a AWS server and used sockets for stable connection to the server
  • We wanted to map out the Waterloo Campus as a graph and use Dijkstra's algorithm to optimize and perform path-finding. We had to minimize the scope of the map in order to complete this project in time.
